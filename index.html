1. The game will provide you with a random-numbered value and to win you have to match it equally.

  2. How do you match it? You are going to be given 4 different colored rupees:
    1. Green Rupee
    2. Blue Rupee
    3. Red Rupee
    4. Purple Rupee

  3. Each round, each rupee's value is randomly selected from a range between numbers 1-12 and will remain hidden

  4. If you do not match the number because you went past the value or fell short, you lose.
